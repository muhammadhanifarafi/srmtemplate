Cara Menggunakan

Icon-icon di atas menggunakan SVG inline dari Heroicons. Copy kode SVG dan paste ke dalam HTML Anda. 

<!-- Contoh penggunaan icon -->
<svg class="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 6v6m0 0v6m0-6h6m-6 0H6"></path>
</svg>

<!-- Dengan warna -->
<svg class="w-5 h-5 text-primary" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 6v6m0 0v6m0-6h6m-6 0H6"></path>
</svg>

Tips: Gunakan class w-5 h-5 untuk ukuran standar. Sesuaikan dengan w-6 h-6, w-8 h-8 untuk ukuran lebih besar. Warna dapat diatur dengan class Tailwind seperti text-primary, text-red-600, dll.
