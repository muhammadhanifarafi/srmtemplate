# Template Preview - Dokumentasi Komponen UI

Folder ini berisi dokumentasi dan preview dari semua komponen UI yang digunakan dalam template.

## Cara Menggunakan

1. Buka file `index.html` di browser untuk melihat halaman beranda
2. Navigasi melalui menu sidebar untuk melihat berbagai komponen:
   - **Buttons** - Variasi tombol dengan berbagai style
   - **Forms** - Elemen form (input, select, checkbox, dll)
   - **Icons** - Koleksi icon SVG yang tersedia
   - **Cards** - Komponen card untuk konten
   - **Tables** - Tabel data dengan berbagai variasi
   - **Modals** - Dialog dan popup modal
   - **Alerts** - Notifikasi dan alert
   - **Typography** - Tipografi dan teks
   - **Colors** - Palette warna yang digunakan

## Fitur

- ✅ Preview semua komponen UI
- ✅ Dark mode support
- ✅ Kode contoh untuk setiap komponen
- ✅ Responsive design
- ✅ Dokumentasi lengkap dengan contoh penggunaan

## Teknologi yang Digunakan

- TailwindCSS untuk styling
- Alpine.js untuk interaktivitas
- SweetAlert2 untuk notifikasi
- SVG Icons (Heroicons style)

## Akses

Buka di browser: `http://localhost/template-preview/index.html` (sesuaikan dengan URL project Anda)
